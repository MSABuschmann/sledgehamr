Detailed Description

This class ensures the uniqueness of a grid by reducing the problem to sets of touples. Each core can work independently on its own UniqueLayout which can then be efficiently merged across cores and nodes to a global unique grid. Memory consumption scales with the number of boxes added, not with the potential number of boxes.
